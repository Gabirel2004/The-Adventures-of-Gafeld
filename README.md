#  The Adventures of Gafeld (As Aventuras de Gafeld)

Este é um jogo de aventura point-and-click feito em Python com a biblioteca **Pygame Zero**. O objetivo é ajudar o Gafeld a chegar na lasanha, desviando dos inimigos.

Este projeto foi criado para um teste de tutoria de Python, seguindo todos os requisitos, como:
* Uso exclusivo de PgZero, math e random
* Menu principal funcional
* Classes próprias para Player e Enemy
* Animação de sprites (parado e andando)

## Como Rodar o Jogo

Para jogar, você precisa ter o Python e o `pgzero` instalados.

1.  **Instale o Pygame Zero:**
    ```bash
    pip install pgzero
    ```

2.  **Rode o jogo:**
    Execute o arquivo `main.py` usando o `pgzrun`.
    ```bash
    pgzrun main.py
    ```

##  Controles

* **Mouse:** User as setas do teclado para mover o Gafeld.
